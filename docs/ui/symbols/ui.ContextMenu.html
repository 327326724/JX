类 详解
ui.ContextMenu(option)
功能:
【ContextMenu】 triggers 参数是指触发右键菜单显示的dom, 右击该节点就显示菜单 也可以不使用这个属性, 调用的地方自己控制显示隐藏菜单
参数:
{Object} option
构造参数 { id: '',//random name: id, container: document.body, className: '' , width: null, triggerEvent: 'contextmenu', triggers: null,//HTMLElement数组 ,触发右键菜单的dom items: [] }
Since:
version 1.0

功能:

参数:
{Object} option
{
title: 鼠标tips
text: 菜单项的文字
link: 指向链接
icon: 图标的配置{ url, style, className }
enable: 是否启用
onClick: 点击处理函数
argument: 点击处理函数接收的额外参数
}
